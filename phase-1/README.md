# Phase 1: SIEM Infrastructure & Access Setup

## Objective
The goal of Phase 1 was to deploy the Wazuh SIEM backend and establish secure access to the dashboard while developing a clear understanding of SIEM architecture and agent workflows.

---

## Environment
- Ubuntu Server for SIEM backend
- VirtualBox virtualization
- NAT networking with secure access methods
- Kali Linux planned for agent deployment

---

## Architecture
Phase 1 focused on deploying:
- Wazuh Manager
- Wazuh Indexer
- Wazuh Dashboard

All components were deployed on a dedicated server to reflect real-world SIEM architecture.

---

## Implementation Summary
- Installed and validated all core Wazuh services
- Verified inter-component communication
- Configured secure dashboard access
- Explored agent registration and authentication workflows
- Performed extensive troubleshooting using logs and service status checks
