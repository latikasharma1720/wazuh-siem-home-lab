# Wazuh SIEM Home Lab

## Overview
This repository documents a multi-phase cybersecurity home lab focused on deploying and understanding a Security Information and Event Management (SIEM) platform using Wazuh.

The project emphasizes SIEM architecture, secure access, agent onboarding workflows, and security monitoring fundamentals.

---

## Project Structure

- **Phase 1:** SIEM Infrastructure Deployment & Dashboard Access  
- **Phase 2:** Endpoint Agent Deployment & Security Event Monitoring (In Progress)

---

## Technologies Used
- Wazuh SIEM
- Ubuntu Server
- Kali Linux
- VirtualBox
- Linux system administration
- SSH-based secure access

---

## Learning Objectives
- Understand SIEM backend architecture
- Deploy and validate core SIEM services
- Explore agent lifecycle and onboarding workflows
- Build troubleshooting skills relevant to SOC environments
